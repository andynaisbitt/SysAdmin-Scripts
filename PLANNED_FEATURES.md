# Planned Features

This document outlines the planned features and new scripts for the SysAdmin-Scripts project.

## Active Directory

*   **Find and Disable Inactive Computer Accounts:** A script to identify and disable computer accounts that have not been active for a specified period of time.
*   **Domain Controller Status Report:** A script to generate a report on the status of all domain controllers in the environment, including replication status, and Directory Service event logs.

## Azure

*   **Azure Resource Management:** A set of scripts for managing Azure resources, including:
    *   Creating and managing virtual machines.
    *   Managing storage accounts and blobs.
    *   Creating and managing virtual networks and subnets.

## Microsoft 365

*   **Mailbox Size and Usage Report:** A script to generate a report on mailbox sizes and usage in Exchange Online.
*   **SharePoint Online Site and Permissions Management:** A set of scripts for managing SharePoint Online sites and permissions, including:
    *   Creating and deleting sites.
    *   Managing site permissions.
    *   Reporting on site usage.

## Security

*   **Security Misconfiguration Checker:** A script to check for common security misconfigurations, such as:
    *   Weak password policies.
    *   Missing security updates.
    *   Open firewall ports.
*   **Server Hardening Automation:** A script to automate the process of hardening servers based on a predefined security baseline.

## Server

*   **Comprehensive Server Health Check:** A script to perform a more comprehensive server health check, including:
    *   Checking for running services.
    *   Monitoring disk space usage.
    *   Analyzing event log errors.
    *   Checking for pending reboots.
